# 簡易待辦事項清單

一個使用純 HTML、CSS 和 JavaScript 開發的待辦事項清單應用。

## 功能特點

- 新增待辦事項
- 標記完成/未完成
- 完成事項自動移至底部
- 記錄創建和完成時間
- 可為完成事項添加備註
- 本地數據儲存
- 密碼保護的重置功能

## 使用方法

1. 直接在瀏覽器中打開 `index.html`
2. 在輸入框中輸入待辦事項，按 Enter 或點擊「添加」按鈕
3. 點擊核取方塊可標記完成/未完成
4. 完成的事項可添加備註
5. 右下角重置按鈕可清空所有事項（密碼：yeahyeah）

## 技術說明

- 純 HTML、CSS 和 JavaScript 實現
- 使用 localStorage 實現數據持久化
- 響應式設計，支援各種設備
